Natural Language Dates

For the attributes of the Nitobi Calendar that expects date values, you can provide a limited number of natural language dates. Or, if you have the Date JS library (http://www.datejs.com) included in your page, you can use much more complex date expressions.

Description

By default, the Calendar component supports the following date expressions:
  • today
  • yesterday
  • tomorrow
  • last week
  • next week
  • last month
  • next month
  • last year
  • next year
But if you include the Date JS Library in your page, you can use more complex date expressions. For example:
  • next Thursday
  • day after tomorrow
  • two weeks from now
More information on the Date JS library can be found at http://www.datejs.com
