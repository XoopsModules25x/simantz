Standalone Components

With the Nitobi Calendar, you can use it as either a standalone input field or as a standalone calendar.

Description

Using the Nitobi Calendar as either an input or a calendar will still give you all the features of each invididual part. The input field will still provide date masking for display and for input. And the Calendar will still support multimonth view, quick nav, etc.
