This sample demonstrates how to create nested tabs. Two tabstrip declarations are place on the page. The first tabstrip is the master, and its second tab references the second tabstrip. Each tab is loaded ondemand. On load, only the first tabstrip is ininitialized.

Note how after the first tabstrip has loaded, it will automatically include, and initialize the second (innermost) tabstrip. The CSS class chosen for the innermost tabstrip is designed to fit squarely into the outer tabs body. You can of course, pick any CSS class for either tabstrip.
