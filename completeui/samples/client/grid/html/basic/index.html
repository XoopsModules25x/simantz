Getting started with a basic grid involves just a little preparation of your page.

  • Set up your Namespace: Your HTML tag should look like this: <html xmlns:ntb="http://www.nitobi.com">
  • Include the Grid Stylesheet: Choose which grid theme you would like to use and include its nitobi.grid.css file in your page
  • Include the Grid JavaScript: Include the nitobi.grid.js javascript file in your page.
  • Put a Grid Tag on the Page
  • Instantiate the Grid: Subscribe a call to nitobi.loadComponent('<YOUR GRID ID>'); to the onload event for the web page.
