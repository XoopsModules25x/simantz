It's quite simple to mix-and-match both remote and static datasources in Grid V3. For example, you can use a database to populate the table, but a static (local) dataset for listboxes and lookup cells. These demos show how static datasources can be encorporated into your page.

Binding Grid to a Static Table

This example uses a static dataset of about 150 rows. The data is embedded directly on the page using a declarative markup. The same technique can be used in each of the available modes (ShowAll, Paging, and LiveScrolling).

This button will loop through the rows and display the dataset in an alert box.
