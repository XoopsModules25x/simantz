Getting started with a basic grid involves just a little preparation of your page.

  • Set up your Namespace <html xmlns:ntb>
  • Include the Combo Stylesheet
  • Include the Combo JavaScript - nitobi.combo.js
  • Include the Combo Language Pack (optional)
  • Put a Combo Tag on the Page
  • Write a Data Handler
  • Instantiate the Combo - nitobi.initCombo('cmbCustomers');
