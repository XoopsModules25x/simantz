SimEDU Management System is an active developed integrated web-based university / college management system. It is a Malaysia leading institute software which assist IPT manage their data easier and faster.
    Modules List:
  • Setting and Maintenance Module
  • Student Affair Module
  • Student Portal
  • Student Online Registration Module
  • Lecturer Portal
  • Finance Module
  • Hostel Module
  • Academic Module
  • Human Resource Module with Payroll System
  • Inventory Module
  • Internal Forum
    Setting and Maintenance Module:
  • User ID and password control.
  • User group control.
  • Control permission of using by user group.
    Student Affair Module:
  • Control and manage student and internet account information.
  • Update student course.
  • Manage student discipline issues data.
  • Manage student graduation and termination information.
  • Student industrial training management.
  • Sending email and SMS to student in bulk.
  • Checking student complaint.
    Student Portal:
  • Update student basic information.
  • Password changing.
  • Register or withdraw subjects.
  • View course or program time table.
  • Checking outstanding fees.
  • View exam result.
  • Request transfer of credits.
  • Requisition of course changing.
  • Application of practical.
  • Complaint of course or institute infrastructure.
  • Application or changing of hostel.
    Student Online Registration Module:
  • Course or Program offering.
  • Program Application by using online or offline by student with upload result copies or relevant certificates function.
  • Automatic applicant filtration base on minimum requirements in the course.
  • Application status informing by generating letter, SMS and email from the system.
  • Generate user ID and password for student.
  • Register student information in the course or program.
    Lecturer Portal:
  • View subject and exam time table.
  • Key in student exam result.
  • Key in student attendance.
  • Upload electronic document (note or question) for student.
  • View pay slip.
    Finance Module:
  • Full set accounting functions:
    Double entry, Duplicate Journal Entry, AP/AR/General Ledger/ Trial Balance, Income Statement (P&L), Balance Sheet, Chart Of Accounts, Lock financial year/month, issue debit/credit note, payment voucher
  • Generate outstanding payment report.
  • Generate payment receipt.
  • Sending reminder letter/SMS/email to student with outstanding payment.
  • Generate cash receipt daily report.
  • Student with outstanding payment not allowed to take for exam except approved by admin.
  • Generate invoice every semester base on registered program.
    Hostel Module:
  • Control and manage student data in hostel by semester.
  • Distribute and update student changing room data. The information will update to academic and account department.
    Academic Module:
  • Linking course to faculty.
  • Setup course fees and credits hour.
  • Determine subjects to particular course.
  • Published course content into website.
  • Upload relevant documents for internet user.
  • Determine minimum requirements for particular course.
  • Offer subjects and determine subject credits hour.
  • Generate subject and exam time table.
  • Assigned lecturer to the subject.
  • Determine lecture room.
  • Online registration of subject by student.
  • Control student attendance.
  • Control exam result.
  • Generate exam result slip.
  • Generate student attendance check list.
    Human Resource Module:
  • Control and manage lecturer and staff data.
  • Control and manage salary, promotion and increment information.
  • Calculate monthly salary.
  • Integrate access card machine date and system.
  • Calculate OT, late in and leave automatic.
  • Generate staff attendance report.
  • Generate CSV file for bank process.
    Inventory Module:
  • Control stock for each department.
  • Generate purchase requisition form.Generate on hand stock report, stock out report and outstanding stock report.
    Internal Forum:
  • Manage by department.
  • For exchange idea.
  • Control access permission base on categories.
